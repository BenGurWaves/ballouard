## 5. SEO & ORGANIC TRAFFIC STRATEGY (The "Stupid Easy" System)

**Philosophy:** Build once, rank forever. Zero marketing, zero burnout.

**Mandatory for Every Site (30-60 min setup per site):**

### A. Long-Tail Keyword Domination
- One hyper-specific keyword per site (100-1,000 monthly searches, zero competition)
- Example format: "best AI prompt template for Claude backend GitHub Cloudflare 2026" not "build website"
- Content: 1,500-2,500 words, answer-first (direct paragraph in first 100 words), H2/H3 headings, lists, tables, unique insights + citations

### B. Schema Markup (JSON-LD)
- Generate via Claude: "Create perfect JSON-LD schema for [Article/Product/FAQ/HowTo] using this content: [paste]"
- Types: Article, Product, FAQ, HowTo, LocalBusiness
- Placement: `<head>` of HTML
- Test: Google Rich Results Test
- Result: Stars, prices, FAQ boxes = 20-40% more clicks

### C. Google Search Console (One-Time)
- Add domain (Cloudflare verification = instant)
- Generate XML sitemap via Claude: "Create full XML sitemap for [pages]"
- Submit sitemap
- Request indexing on main page
- Check "Enhancements → Structured data" for rich results

### D. GEO Optimization (AI Overview Capture)
- Structure every page:
  - Direct TL;DR answer at top
  - FAQ section at bottom
  - "Last updated: [today's date]"
  - Clean H2/H3 + bullet lists/tables
- `robots.txt`: Allow GPTBot, ClaudeBot, PerplexityBot
- Goal: Become the cited source in AI Overviews

### E. Directory Listings (Passive Trust)
- 5-10 niche/general directories (Product Hunt alternatives, industry-specific)
- Claude writes descriptions
- Submit once, traffic trickles forever

**Prohibited:** Social media marketing, Reddit spam, DMs, daily posting, ads, backlink outreach.

## 6. DESIGN SYSTEM: QUIET LUXURY MANDATE

**Default Mode for ALL Output:** Quiet Luxury. Never AI Slop.

### Core Philosophy
- Whisper, never shout. Timeless > Trendy.
- Restraint, intentional imperfection (wabi-sabi), obsessive detail, intrinsic quality, human warmth.
- Negative space is a feature, not a bug.
- Inspiration: The Row, Loro Piana, Bottega Veneta, Aesop, The Line, Brunello Cucinelli, 1990s Vogue, Kinfolk, Apartamento.

### The Checklist (Verify Every Output)

| Element | AI Slop (DESTROY) | Quiet Luxury (DELIVER) |
|---------|-------------------|------------------------|
| **Philosophy** | Maximize attention, hype, conversions | Serve the product. Let quality speak. |
| **Colors** | Bright, saturated, gradients, purple/blue | Muted neutrals: warm ivory, charcoal, taupe, soft greige, deep olive, black, cream. One subtle accent only. |
| **Typography** | Inter, Roboto, too many fonts, bold everything | 1-2 faces max. Elegant sans (Neue Haas Grotesk, Satoshi, PP Mori) + refined serif (Recoleta, Cormorant). Leading 1.6-2.0, massive whitespace. |
| **Layout** | Hero + 3-column grid + CTA everywhere | Editorial asymmetry or perfect balance. Massive whitespace. Single focus per section. Scrolling = turning pages in $300 coffee-table book. |
| **Imagery** | Stock AI women, perfect skin, Midjourney hands, oversaturated | Editorial photography: natural light, film grain, subtle texture, real humans (imperfect beauty), product close-ups showing material quality. |
| **Copy Tone** | Enthusiastic, list-heavy, "you deserve", emojis | Restrained, confident, poetic but concise. Short sentences. Zero hype. Private concierge voice. |
| **Details** | Perfect symmetry, rounded corners everywhere | Micro-imperfections: slight organic curves, hand-finished edges, custom hover states, subtle paper texture on buttons. |
| **Animations** | Bouncy reveals, parallax overload | Restrained: gentle fade-ins, smooth scroll-triggered parallax at 0.2 speed, or none. |
| **Vibe** | Busy, modern startup energy | Calm gallery / private members club / old European atelier |

### Website Specifics
- **Hero:** 80-90% image/video + 1-2 lines text max. No "Welcome" or "Revolutionary AI Tool."
- **Navigation:** 4-6 links max. Logo + "Shop" + "Journal" + "About." Hamburger mobile-only.
- **Footer:** Tiny, elegant, no 47 links.
- **Speed:** Lighter than air, yet rich.
- **Mobile:** Same elegance, never cramped.

### Copywriting Rules
- **Banned forever:** delve, embark, tapestry, realm, unlock, elevate, game-changer, seamless, "In today's fast-paced world..."
- **Replace with:** Quiet observation, precise language, sensory details.
- **Length:** 30-50% shorter than default AI output. Every word earns its place.
- **Example Slop:** "Unlock the future of productivity with our revolutionary AI tool."
- **Example Quiet Luxury:** "A place where thoughts settle. Tools that disappear until needed."

### Red Flags — Regenerate Immediately If Present
- More than one exclamation mark per page
- Any emoji
- "In today's fast-paced world..."
- Bullet points longer than 8 words without poetic rhythm
- Perfect 50/50 symmetry everywhere
- Generic hero text bigger than 60px

### Internal Prompt Prefix (Auto-Applied)
> "Generate in strict Quiet Luxury mode using the Claude.md guidelines. Output only the final piece. Zero explanations unless asked. Destroy all AI slop markers. Make it feel hand-crafted by a reclusive European atelier in 2026."
